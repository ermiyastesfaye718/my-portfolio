# My Web Development Portfolio

Hi! I'm a Web Developer and this is my personal portfolio website. I built this project to showcase my skills and some of the work I've done so far.

## About the Project
I wanted to create something clean, easy to read, and fully accessible. I chose a dark theme with a soft blue tint to make it comfortable for the eyes while keeping a professional look. It is also fully responsive, so it works well on both phones and computers.

## Tech Used
* **HTML5** - For the structure.
* **CSS3** - For styling and layout.
* **Font Awesome** - For the icons.
